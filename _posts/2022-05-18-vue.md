### npm
node와 vscode설치가 완료되었다면 다음과 같은 방법으로 환경설정을 한다.<br>
우선 visual studio code에서 터미널을 연 뒤 vue cli[vue개발환경 세팅을 도와주는 프로그램]를 설치한다.<br>
명령어는 다음과 같다.<br>
```cmd
npm install -g @vue/cli
```

### 설치 에러
npm오류 > node.js설치를 다시 하거나, yarn 1.22을 설치할것 설치 후 재부팅 필요

yarn 설치 후는 명령어가 다음과 같다.
```cmd
yarn global add @vue/cli
```
