## node.js란?
간단히 설명하여 [자바스크립트 패키지 매니저]이다.<br>
자바스크립트 패키지 매니저란, 프로젝트가 의존하고 있는 패키지를 효과적으로 설치, 갱신, 삭제할 수 있도록 도와주는 관리 도구 이다.
예를들어 리액트던, 뷰던, 웹팩이던 자바스크립트 기반의 프레임워크를 사용하고 싶다면 그와 관련된 패키지에 대한 설치가 필요한데 이때 설치해 줄 수 있도록 만들어주는 관리 도구라고 보면 된다.

### npm
node package manager의 약자로, 설치하고자하는 파일이 이미 기존에 존재하는지 확인이 불가능하며, 한번 설치했던 모듈을 업데이트 하지 않는다. 따라서 개발자 입장에서는 최신 모듈이 업데이트 될때마다 번거롭게 업데이트를 해줘야하는 단점이있다.

### npx
node package execute의 약자로, 기존 경로에 설치하고자하는 파일이 이미 있는지를 확인하여 설치되어 있다면 실행, 설치되어 있지 않다면 새롭게 설치를 진행하는 역할을 하고, 설치된 모듈이 최선버전인지를 확인하여 업데이트 해 준다.

npm의 단점을 보완하는 것이 npx로 자바스크립트 패키지 설치시 유용하다.

### yarn
기존 node.js가 가지는 npx, npm의 단점을 보안 및 설치 속도를 훨씬 빠르게 할 수 있는 패키지 매니저이다.
