## vue지시자
### v-bind
데이터 바인딩을 사용하면, javascript의 변수를 html자체에 꽂아넣을 수 있게 해준다. <br>
기존에 특정 html에 값을 넣으려고 하면 다음과 같이 해야했다.
```js 
var name = "annie";
document.getElementsByClassName('h2').innerText = `내이름은 ${name}이다.`;
```
그러나, 뷰에서는 다음과 같이 간단하게 사용할 수 있다. <br>
우선, vue instance부분에 아래와 같이 data영역을 추가한다.<br>
data영역에 추가된 object형 자료들은 앞으로 vue 문서 안에서 data값으로 사용될 수 있다.
```js
export default {
  name: 'App',
  data(){
    return{
    
    }
  },
  components: {
    
  }
}
```

그 다음, 문서 안에서 사용할 변수를 object 형식으로 data 인스턴스안에 추가하면 된다.
```js
export default {
  name: 'App',
  data(){
    return{
        name1 :'고길동',
        name2 :'둘리',
    }
  },
  components: {
    
  }
}
```

위와같이 인스턴스안에 추가된 변수는 아래처럼 vue문서 안에 사용할 수 있다.<br>

### v-for
만약, data instance에 다음과 같은 배열 데이터를 넣었다고 가정해보자.

  name: 'App',

```text

    <img alt="Vue logo" src="./assets/logo.png">

        <button @click="내용">좋아요!</button>
    </div>
    </template>
```
