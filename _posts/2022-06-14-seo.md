## SEO란 ? 
웹 사이트가 포털 검색 결과에 잘 보이도록 최적화 하는 과정을 의미한다.

### title
문서가 검색될때 title요소 내용이 검색결과의 제목으로 표시되기때문에 <br>
title요소는 고유하고 정확한 페이지 제목을 입력해야한다.<br>
따라서 자연스럽게 읽히고 내용의 주제를 효과적으로 전달하는 제목의 텍스트를 선택해야한다.<br>
**title은 각 페이지마다 다른 설명을 해주면 좋다.
```html
<html>
<head>
    <title>박정아의 개인 블로그</title>
</head>
<body>
```

### description
설명태그는 아래와 같이 meta태그로 사용하는데, 페이지 내용을 요약해서 포털사이트에 제공한다.
페이지 제목과 다른점은 제목의 경우 간단한 한 문장으로만 쓸수있지만, 설명태그의 경우 한 단락을 쓸 수 있다. <br>
해당 설명내용은 키워드로만 채우지 않도록 주의한다.

```html
<html>
<head>
    <title>박정아의 개인블로그</title>
    <meta name="description" content="html, css, javascript, jquery, sass 및 SPA페이지 제작 전문입니다. 또한 가끔 강의도합니다.">
</head>
```
