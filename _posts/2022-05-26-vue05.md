## Vue LifeCycle
<img src="https://joshua1988.github.io/vue-camp/assets/img/lifecycle.dcbe29f6.png" />
<sub>출처: https://joshua1988.github.io/</sub> <br>

vue가 생성될때 몇단계의 과정을 거치게되는데 이를 vue의 라이프사이클이라고 한다. <br>
즉, vue인스턴스 생성 후 우리 눈에 보여지고 변경되고 사라지기까지의 단계를 말한다.

### beforeCreate
뷰 생성 후 가장먼저 실행되는 단계. vue인스턴스가 초기화된 직후에 발생한다. data event, watcher, 컴포넌드 로드등의 단계의 전 단계이기때문에 인스턴스 중 data와 methods영역에 접근할 수 없다.

### created
data인스턴스 및 methods, watcher가 활성화 되는 단계이지만, 아직 dom이 로딩되기 전 단계이다.

### beforeMount
dom로딩 직전에 호출되는 단계로, dom이 생성되어 있기는하나 정확히 로딩되지는 않은 단계를 의미한다.

### mounted
가상 dom이 실제로 로딩이 완벽하게 된 상태이므로, 인스턴스의 모든 요소에 접근이 가능하다.

### beforeUpdate
컴포넌트의 data값이 변경된다고 할때, 그 data값을 해당 dom에 적용시켜야한다. 이러한 변화 직전에 호출되는 단계로, 변할값을 이용하여 작업할 수 있다.

### updated
dom에 실제로 data값이 변경되어서 적용될때의 단계를 의미하며, 변경된 값을 통하여 dom에 접근하고 싶을때 가장 적절한 단계이다. 그러나, 이 훅에서 data를 변경하는것은 무한루프를 일으킬 수 있으므로 이 훅에서는 데이터를 바꾸기에 적합하지 않다.

### beforDestroy
인스턴스가 해체되기전 호출되는 훅으로, 완벽히 해체되기 이전이기때문에 인스턴스 속성에 접근이 가능하다. 이 단계에서는 인스턴스가 사라지기전에 해야할 일을 처리하면 된다.

### destryed
인스턴스가 완벽히 해체되고 난 직후이다. 이때는 인스턴스의 모든 속성에 접근할 수 없다.


## 사용방법

### 페이지 로딩이 된 후 1초마다 특정 값 추가하기
구현방법은 아래와 같이 mounted인스턴스에 함수나 원하는 데이터 변경값을 추가하면된다. 여기서 주의해야할 점은 함수 작성시 function(){}형태로 작성하면 this가 데이터가 아닌 dom을 가르키게 되기 때문에 무조건 화살표 함수를 써야한다. <br>

인스턴스의 모든 내용들은 다 object형태로 들어가 있기때문에 instance안에서 활용할 data들은 모두 this를 앞에 붙여야한다. 
```vue
<template>
    <p>당신이 접속한지  { {second} }초 지났습니다.</p>
</template>
<script>
    export default {
        name: 'App',
        data(){
            second:0,
        },
        mounted(){
            setInterval(()=>{
                this.second++;
            });
        }
    }
</script>
```

### 초수마다 분 계산하기
second에 대한 data값이 변할때마다 그 변한값을 60으로 나누어 min에 대입하면 된다.
이때, updated를 쓰면 무한루프가 걸릴 수 있으니 beforeUpdate를 쓰는것이 좋다.

```vue
<template>
    <p>당신이 접속한지  { {second} }초 지났으며. 해당 시간은{ {min} }분 입니다.</p>
</template>
<script>
    export default {
        name: 'App',
        data(){
            second:0,
            min:0,
        },
        mounted(){
            setInterval(()=>{
                this.second++;
            });
        },
        beforeUpdate(){
            this.min = parseInt(this.second / 60);
        }
    }
</script>
```
